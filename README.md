# Claude Code - Mobile UX

A mobile-optimized user interface that replicates the Claude Code experience for mobile devices.

## Features

- **Dark Theme**: Authentic Claude Code dark theme with orange accents
- **Responsive Design**: Fully responsive layout optimized for mobile and tablet devices
- **Chat Interface**: Interactive chat-style conversation interface
- **Code Blocks**: Syntax-highlighted code blocks with copy functionality
- **Sidebar Navigation**: Collapsible sidebar for chat history (mobile-friendly)
- **Touch Optimized**: All controls are touch-friendly and optimized for mobile interaction
- **Auto-resizing Input**: Message input that automatically expands as you type
- **Typing Indicators**: Visual feedback when assistant is "thinking"

## Design Highlights

### Color Palette
- Primary Background: `#1a1a1a`
- Secondary Background: `#252525`
- Text Primary: `#e8e8e8`
- Accent Orange: `#ff8c42`

### Typography
System font stack for optimal performance and native feel across all platforms.

### Components
- **Header**: Fixed navigation with menu toggle and new chat button
- **Sidebar**: Slide-out navigation with chat history
- **Messages**: Chat bubbles with distinct styling for user and assistant
- **Code Blocks**: Styled code display with copy-to-clipboard functionality
- **Input Area**: Bottom-fixed input with attachment and send controls

## Usage

Simply open `index.html` in a web browser to view the mobile UX interface.

### Keyboard Shortcuts
- **Enter**: Send message
- **Shift + Enter**: New line in message

## Technical Details

- Pure HTML, CSS, and vanilla JavaScript
- No external dependencies
- Mobile-first responsive design
- CSS animations for smooth interactions
- Optimized scrolling and performance

## Browser Compatibility

Works on all modern browsers including:

- Safari (iOS and macOS)
- Firefox (latest)
- Mobile browsers (iOS Safari, Chrome Mobile)
